# Integrations

Integration guides cover service-specific setup for connecting Essentials features with third-party platforms.

Use these guides when a service requires provider-side account setup, custom app credentials, OAuth permissions, or other steps beyond the generic [Auths, Keys & Secrets](../auths/) workflow.

## Dynamic Content

- [Facebook](./facebook) - Set up a custom Facebook developer app for Facebook and Instagram Business sources.
- [Google](./google) - Set up a custom Google developer app for Google sources and actions.
- [Instagram](./instagram) - Set up Instagram personal and business account access for the Instagram Source.
- [LinkedIn](./linkedin) - Set up a custom LinkedIn developer app for the Dynamic Content LinkedIn Source.
- [Cloudflare Stream](./cloudflare-stream) - Configure Cloudflare API access for the Cloudflare Stream Source.
- [TikTok](./tiktok) - Connect a TikTok account for the TikTok Source.
- [Vimeo](./vimeo) - Connect a Vimeo account for the Vimeo Source.
- [YouTube](./youtube) - Connect a Google account or API key for the YouTube Source.
- [Twitter](./twitter) - Set up a custom Twitter/X developer app for the Dynamic Content Twitter Source.

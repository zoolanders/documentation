# AcyMailing Integration

    <!--@include: ./assets/brands/acymailing.svg-->

The AcyMailing integration is available through [AcyMailing Action](../addons/forms/actions/acymailing). Use it to subscribe or unsubscribe members from AcyMailing lists.

For setup details, list selection, and custom field mapping, see the [AcyMailing Action](../addons/forms/actions/acymailing) docs.

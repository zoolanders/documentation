# Airtable Integration

<div class="tm-resource-icon">
    <!--@include: ./assets/brands/airtable.svg-->
</div>

The Airtable integration is available through [Airtable Source](../addons/dynamic/sources/airtable) and [Airtable Action](../addons/forms/actions/airtable). Use it to read from Airtable bases or create, update, and delete records from forms.

Connect an Airtable account, then choose the base and table you want to work with.

For setup details, field mappings, and record behavior, see the [Airtable Source](../addons/dynamic/sources/airtable) and [Airtable Action](../addons/forms/actions/airtable) docs.

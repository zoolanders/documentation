# Brevo Integration

<div class="tm-resource-icon">
    <!--@include: ./assets/brands/brevo.svg-->
</div>

The Brevo integration is available through [Brevo Action](../addons/forms/actions/brevo). Use an API key to create, update, delete, or double opt-in contacts in Brevo lists.

For setup details, audience mapping, and the available contact actions, see the [Brevo Action](../addons/forms/actions/brevo) docs.

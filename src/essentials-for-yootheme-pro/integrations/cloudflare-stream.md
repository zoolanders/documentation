# Cloudflare Stream Integration

<div class="tm-resource-icon">
    <!--@include: ./assets/brands/cloudflare.svg-->
</div>

The Cloudflare Stream integration is available through [Cloudflare Stream Source](../addons/dynamic/sources/cloudflare-stream). Use a Cloudflare account, API token, and optional signing key to fetch videos from Stream.

For setup details, query options, and video field mappings, see the [Cloudflare Stream Source](../addons/dynamic/sources/cloudflare-stream) docs.

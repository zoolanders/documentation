# YouTube Integration

    <!--@include: ./assets/brands/youtube.svg-->
</div>

The YouTube integration is available through [YouTube Source](../addons/dynamic/sources/youtube). Use a Google account OAuth connection for channel and playlist sources, or an API key for public video sources.

For setup details, account requirements, and the available source queries, see the [YouTube Source](../addons/dynamic/sources/youtube) docs.
